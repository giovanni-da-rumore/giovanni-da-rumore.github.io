Projects

Cicadas Song

Named after a Platonic Myth and modeled off Genius, Cicadas Song is a digital library that allows users to read, upload and annotate texts. The site runs primarily on a Backbone.js front end, straddling a RESTful Rails API. It uses jQuery to create annotations, stores image uploads (through urls or files) on AWS, and supports multi-table lookup through PgSearch.

Ancient Asteroids

Ancient Asteroids is a spin off the classic arcade game of similar name, but with Ancient Greek Ruins for levels and Ancient Greek vase paintings for asteroids. It runs primarily through Javascript and jQuery and uses HTML5 Canvas for illustration.

Chess

A simple chess game to be played in the terminal. The game is built in Ruby, uses Colorize and UTF-8 characters for an aesthetically pleasing interface. It also uses polymorphism and class inheritance for DRY code.
